# Terminal Text Buffer

A high-performance, memory-efficient terminal text buffer implementation in Kotlin. This project was developed as a technical assessment for the JetBrains internship program.

## 🚀 Overview

This library provides a robust engine for managing a terminal's text buffer, including text insertion, cursor navigation, and scrollback management.

## 🛠️ Requirements

- **JDK 17** or higher
- **Gradle** (included via Gradle Wrapper)


## 🏗️ Build & Setup

Since this is a library project, there is no "main" execution point. The correctness and behavior are verified through a comprehensive suite of unit tests.

### Compile and Build
To compile the project and ensure the build script is correct, run:
```bash
./gradlew build
```

### Test
```bash
./gradlew test
```
